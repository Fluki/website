Setting up Caps Lock as a Esc and Mod key

#increase key speed via a rate change
xset r rate 300 50
#Map the caps lock to super...
setxkbmap -option caps:super
#When is pressed only once, treat it as escape
killall xcape 2>/dev/null ; xcape -e 'Super_L=Escape'
#Map the neby button to right super as well.
xmodmap -e 'keycode 135 = Super_R'
#Turn off the caps lock if on since there is no longer a key for it
xset -q | grep "Caps Lock:\s*on" && xdotool key Caps_Lock

Generating and adding SSH to GitHub

Generating SSH key
ssh-keygen -t ed25519 -C email@.com
Starting SHH agent
eval "$(ssh-agent -s)"
Adding private key to ssh agent
vim ~/.ssh/config
Host *
AddKeysToAgent yes
IdentityFile ~/.ssh/new_created_key
ssh-add ~/.ssh/new_created_key
Adding SSH key to GitHub
Go to
In key field just put newly created key.pub

!!! Every time you change SSH user you must go and do the following command

ssh-add ~/.ssh/wanted_key
&&
eval "$(ssh-agent -s)"

Setting up imwheel & adding it to GNOME autostart

After installing imwheel program, you should create
.imwheelrc
in your home directory
After that you should edit the .imwheelrc file.
I wont go into great detail explaining how and what each line does.
But changing the number values changes the scroll behaviour
".*"
None, Up, Button4, 4
None, Down, Button5, 4
Shift_L, Up, Shift_L|Button4, 4
Shift_L, Down, Shift_L|Button5, 4
Control_L, Up, Control_L|Button4
Control_L, Down, Control_L|Button5
In order to make Gnome session autostart this imwheel program,
you can create imwheel.desktop file in your .config/autostart/
I eddited my file like this:
[Desktop Entry]
Type=Application
Exec=imwheel -b 45
Terminal=true
Hidden=false
X-GNOME-Autostart-enabled=true
Name=imwheel
Comment=change scroll speed
Where -b 45 flags tell the program that we want to use back and foward buttons on our mouse.
